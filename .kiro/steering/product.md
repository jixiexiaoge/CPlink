# CPlink - 车载导航应用

CPlink是一个Android车载导航应用，专为电动汽车和智能驾驶场景设计。该项目包含Android客户端和Web管理后台两个主要组件。

## 核心功能

- **Android导航应用**: 基于Jetpack Compose的现代化车载导航界面
- **视频播放**: 集成ExoPlayer支持车载视频内容播放
- **用户数据管理**: 跟踪使用统计、行驶距离、使用时长等数据
- **Web管理后台**: Flask驱动的用户数据分析和管理平台
- **API接口**: 提供用户注册、数据更新、视频管理等RESTful API

## 用户等级系统

- **新用户 (1)**: 首次注册用户
- **支持者 (2)**: 支持项目的用户  
- **赞助者 (3)**: 赞助项目的用户
- **铁粉 (4)**: 忠实粉丝用户
- **管理员 (-1)**: 管理员专用等级

## 技术特色

- 企业级APK加密保护 (ProGuard/R8混淆)
- 数字签名和安全配置
- 实时数据同步和分析
- 支持多种车型的兼容性测试
